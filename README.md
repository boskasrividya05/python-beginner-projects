# Python Beginner Project: Calculator
This is a simple calculator program using Python.
It can perform Addition, Subtraction, Multiplication, and Division.
## How to Use
1. Run the program.
2. Enter first number when prompted.
3. Enter second number when prompted.
4. The program will display Addition, Subtraction, Multiplication, and Division results.
## Sample Output
Enter first number: 10
Enter second number: 5
Addition = 15
Subtraction = 5
Multiplication = 50
Division = 2.0

Python
